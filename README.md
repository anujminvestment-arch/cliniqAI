# CliniqAI

AI-powered clinic management and patient communication platform.

## Overview

CliniqAI is a multi-tenant SaaS platform that enables doctors and clinics to automate patient communication, appointment scheduling, queue management, and follow-up interactions using voice AI and intelligent automation.

## Features (MVP)

- **Multi-Tenant Clinic System** — Each clinic operates independently with isolated data
- **Appointment Management** — Create, edit, cancel, reschedule with multi-doctor support
- **Queue Tracking** — Real-time queue length, wait time, and position
- **AI Call Booking** — Voice AI receptionist for phone-based appointment management
- **Doctor Dashboard** — Centralized management across multiple clinic locations
- **Patient Portal** — Appointments, records, prescriptions, invoices, queue status
- **Billing System** — Invoices, payments, receipts, transaction history
