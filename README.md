# CliniqAI

AI-powered clinic management and patient communication platform.

## Overview

CliniqAI is a multi-tenant SaaS platform that enables doctors and clinics to automate patient communication, appointment scheduling, queue management, and follow-up interactions using voice AI, WhatsApp chatbot, and intelligent automation. Built AI-first for small clinics in India.

## Why CliniqAI?

- **AI-first** — Voice + WhatsApp + web chat as core, not an afterthought
- **WhatsApp-native** — Patients book, check queue, pay, and get prescriptions all via WhatsApp
- **Small clinic focus** — Built for solo doctors and 2-5 doctor clinics in Tier 2/3 cities
- **Zero-setup** — Sign up, configure hours, AI starts answering calls in 30 minutes
- **Cheaper than a receptionist** — Replaces ₹12-18K/month manual processes

## Features

### Phase 1: MVP+ — "Clinic Can Operate"
- **AI Voice Receptionist** — Answer calls, book appointments, queue status, smart escalation
- **Multi-Tenant Clinic System** — Each clinic as independent tenant with isolated data
- **Appointment Management** — CRUD, multi-doctor scheduling, buffer times
- **Queue Management** — Real-time tracking, no-show detection, priority rules
- **Digital Prescriptions** — Doctor e-signature, WhatsApp delivery, print-friendly
- **Billing & Online Payments** — Razorpay/UPI, invoices, receipts, payment reminders
- **WhatsApp Notifications** — Appointments, queue updates, payments, prescriptions
- **Patient Portal** — Appointments, health records, prescriptions, billing, queue tracking
- **Doctor Dashboard** — Consolidated view across clinics, patient summaries
- **Granular RBAC** — 7+ roles with configurable permissions
- **Compliance** — DPDPA consent, audit logging, data encryption

### Phase 2: Growth — "Better Than Manual"
- **WhatsApp Chatbot** — Full conversational AI on WhatsApp
- **Telemedicine** — Video consultation (Jitsi/Daily/Twilio)
- **Waitlist Management** — Auto-fill cancelled slots
- **Patient Feedback & NPS** — Post-visit surveys, Google Review prompts
- **Multi-Language** — Hindi + regional languages
- **Medication Reminders** — Push + WhatsApp adherence
- **Clinic Analytics** — Revenue, no-shows, utilization, satisfaction benchmarks

### Phase 3: Scale — "Why Clinics Pay Premium"
- **AI Triage Bot** — Pre-booking symptom assessment
- **Family Health Hub** — Multi-member profiles, shared calendar
- **Digital Health Card** — QR-based patient ID
- **Lab Integration** — Test ordering, result delivery
- **Campaign Engine** — Health campaigns, re-engagement
- **Referral Management** — Doctor-to-specialist with tracking

### Phase 4: Moat — "Can't Switch Away"
- **AI Clinical Notes** — Ambient documentation
- **Clinic Marketplace** — Patient discovery
- **Insurance Integration** — Verification, claims
- **Enterprise** — SSO, advanced audit, SLA
