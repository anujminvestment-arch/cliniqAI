# Workflows — CliniqAI

## User Journeys

### Patient — Book Appointment via AI Call
1. Patient calls clinic number
2. AI voice receptionist answers
3. AI identifies patient (new or returning)
4. If new → AI collects registration details
5. AI provides available slots
6. Patient selects slot
7. AI confirms booking
8. Patient receives SMS confirmation

### Patient — Book via Portal
1. Patient logs into patient portal
2. Selects clinic and doctor
3. Views available time slots
4. Books appointment
5. Receives confirmation

### Patient — Check Queue Status
1. Patient calls clinic or opens portal
2. AI/portal shows current queue position
3. Estimated wait time displayed

### Doctor — Daily Workflow
1. Doctor logs into dashboard
2. Views today's appointments across clinics
3. Reviews patient symptom intake summaries
4. Completes consultations
5. Generates invoices
6. Reviews AI follow-up suggestions

### Clinic Staff — Register Walk-in Patient
1. Staff opens registration form
2. Enters patient details manually
3. Patient added to queue
4. Invoice generated after visit
